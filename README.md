#COMP3607_Jeopardy_Project#
Java-based multiplayer Jeopardy game simulation 

How to play:

- Navaigate to "Main.java"
- To play the game, first begin by clicking the "RUN" button on the top right hand side of the API (looks like a play button)
- Next, enter the path of the question file as prompted in the terminal
- Next, enter the number of players and their names 
- Follow through the questions by answering as prompted
- Type "QUIT" at any time to exit the game before it ends.
